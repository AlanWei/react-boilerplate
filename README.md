# React Boilerplate

## Getting Started

### Development

```bash
yarn install
yarn dev
```

Navigate to `http://localhost:8080`

### Build

```bash
yarn build
```

Compiled assets files will be in `build` folder

> For the server side rendering version, please visit [react-boilerplate-ssr](https://github.com/AlanWei/react-boilerplate-ssr)
